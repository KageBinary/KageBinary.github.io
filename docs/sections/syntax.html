Syntax Reference

Understanding the syntax of Pithon is crucial for writing efficient and error-free code. This section provides detailed information on Pithon's syntax, covering variables, data types, control structures, functions, and more.

Variables and Data Types

Pithon supports dynamic typing, allowing variables to hold data of any type without explicit type declarations. 


# Example of variable declaration in Pithon
let x = 10
let name = "Pithon"
let isActive = true

Control Structures

Pithon provides standard control structures such as conditional statements and loops to control the flow of execution.

If Statements


# Example of an if statement in Pithon
let age = 18

if age >= 18:
    display("You are eligible to vote.")
else:
    display("You are not eligible to vote.")

Loops


# Example of a for loop in Pithon
for i in 1..5:
    display("Iteration " + i)

# Example of a while loop in Pithon
let count = 0
while count < 5:
    display("Count is " + count)
    count = count + 1

Functions

Functions in Pithon allow you to encapsulate reusable code blocks. They can accept parameters and return values. 


# Defining a function in Pithon
function greet(name):
    display("Hello, " + name + "!")

# Calling the function
greet("Alice")

Comments

Use comments to annotate your code. Single-line comments start with the `#` symbol. 


# This is a single-line comment

let x = 10  # Variable declaration
