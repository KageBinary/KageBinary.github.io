Interpreter

The Pithon interpreter executes Pithon scripts, enabling developers to run and test their code efficiently. Built using Python and TextX, it offers features such as type inference, comprehensive error handling, and integration with built-in modules.

Installation

Follow these steps to set up the Pithon interpreter on your system:

  1. Prerequisites: Ensure you have Python 3.6 or higher installed. You can download it from the official Python website.
  2. Download Interpreter Files: Visit the Download page and download the latest Pithon interpreter package (e.g., ZIP file).
  3. Extract Files: Extract the downloaded ZIP file to a directory of your choice.
  4. Install Required Libraries: Open your terminal or command prompt, navigate to the extracted interpreter directory, and install the necessary Python libraries by running: 
    
pip install textx matplotlib numpy

Running the Interpreter

To execute a Pithon script, use the following command: 


python pithon_interpreter.py path/to/your_program.pithon

Example: 


python pithon_interpreter.py examples/concat_example.pithon

Key Features

  • Type Inference: Automatically deduces variable types, reducing the need for explicit declarations.
  • Error Handling: Provides detailed error messages to assist in debugging.
  • Built-in Modules: Includes Math, Physics, and System modules with essential constants and functions.
  • Functional Utilities: Supports functions like apply, summation, and factorial for advanced operations.
  • Graphing Capabilities: Integrates with Matplotlib to create and display graphs.

Troubleshooting

If you encounter issues while using the interpreter, consider the following steps:

  1. Ensure all dependencies are installed correctly by verifying with pip list.
  2. Check your Pithon script for syntax errors or unsupported operations.
  3. Refer to the Contact page for further assistance.
