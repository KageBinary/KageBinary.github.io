Built-in Functions

Pithon provides a set of built-in functions that facilitate various operations, from functional programming utilities to mathematical computations. These functions are readily available for use in your Pithon scripts without the need for additional imports.

List of Built-in Functions

  • apply(function, set) - Applies a user-defined function to each element of a set and returns a new set with the results.
  • summation(varName, start, end, expression) - Calculates the summation of an expression over a range of integer values.
  • factorial(number) - Computes the factorial of a non-negative integer.

Function Usage Examples

apply()

The apply function takes a user-defined function and a set as arguments, applies the function to each element of the set, and returns a new set containing the transformed elements. 


# Example: Using apply Function in Pithon

# Define a set
let numbers = {1, 2, 3, 4, 5};

# Define a function that squares its input
let square(x) = x ^ 2;

# Apply the function to each element of the set
let squared_numbers = apply(square, numbers);

# Display the original and squared sets
display("Original Numbers:", numbers);         # Output: Original Numbers: {1, 2, 3, 4, 5}
display("Squared Numbers:", squared_numbers);  # Output: Squared Numbers: {1, 4, 9, 16, 25}

summation()

The summation function calculates the sum of an expression over a range of integer values. It takes a variable name, start and end values for the range, and the expression to be summed. 


# Example: Using summation Function in Pithon

# Define the variable, range, and expression
let total = summation(i, 1, 5, i * 2);

# Display the result
display("Summation Result:", total);  # Output: Summation Result: 30

**Explanation:**

  1. Variable Name: i
  2. Start: 1
  3. End: 5
  4. Expression: i * 2
The summation calculates (1*2) + (2*2) + (3*2) + (4*2) + (5*2) = 30.

factorial()

The factorial function computes the factorial of a non-negative integer. Factorial of a number n is the product of all positive integers less than or equal to n. 


# Example: Using factorial Function in Pithon

# Calculate factorial of 5
let fact = factorial(5);

# Display the result
display("Factorial of 5:", fact);  # Output: Factorial of 5: 120

**Explanation:** factorial(5) computes 5 * 4 * 3 * 2 * 1 = 120.
