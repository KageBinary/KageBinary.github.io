Built-in Functions

Pithon provides a rich set of built-in functions that facilitate various operations, from basic input/output to complex mathematical computations. This section lists all the built-in functions available in Pithon, along with their descriptions and usage examples.

List of Built-in Functions

  • display() - Outputs text or variables to the console.
  • input() - Takes user input from the console.
  • len() - Returns the length of a string or list.
  • range() - Generates a sequence of numbers.
  • math.sqrt() - Calculates the square root of a number.
  • physics.calculate_force() - Computes force based on mass and acceleration.
  • system.get_current_directory() - Retrieves the current working directory.

Function Usage Examples

display()

Outputs text or variables to the console. 


# Example of display function
let message = "Hello, Pithon!"
display(message)  # Outputs: Hello, Pithon!

input()

Takes user input from the console. 


# Example of input function
let name = input("Enter your name: ")
display("Hello, " + name + "!")

len()

Returns the length of a string or list. 


# Example of len function
let text = "Pithon"
let text_length = len(text)
display("Length of text:", text_length)  # Outputs: Length of text: 6

range()

Generates a sequence of numbers. 


# Example of range function
for i in range(1, 5):
    display("Number:", i)
    # Outputs: Number: 1
    #          Number: 2
    #          Number: 3
    #          Number: 4

math.sqrt()

Calculates the square root of a number. 


# Example of math.sqrt function
let number = 16
let square_root = math.sqrt(number)
display("Square root of", number, "is", square_root)  # Outputs: Square root of 16 is 4.0

physics.calculate_force()

Computes force based on mass and acceleration. 


# Example of physics.calculate_force function
let mass = 10  # in kilograms
let acceleration = 9.8  # in meters per second squared
let force = physics.calculate_force(mass, acceleration)
display("Force:", force)  # Outputs: Force: 98.0

system.get_current_directory()

Retrieves the current working directory. 


# Example of system.get_current_directory function
let current_dir = system.get_current_directory()
display("Current Directory:", current_dir)
