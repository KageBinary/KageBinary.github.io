Modules

Pithon comes with a set of built-in modules that extend its functionality, allowing developers to perform a wide range of tasks without writing extensive code. This section explores the available modules, their capabilities, and guidelines for creating custom modules.

Built-in Modules

Below are the built-in modules available in Pithon. Each module provides specific functionalities to aid in development.

Math Module

The Math module offers a collection of mathematical functions and constants, enabling complex calculations with ease. 


import math

let result = math.sqrt(25)
display(result)  # Outputs: 5.0

Physics Module

The Physics module provides functions related to physics calculations, including force, energy, and motion. 


import physics

let force = physics.calculate_force(mass=10, acceleration=9.8)
display(force)  # Outputs: 98.0

System Module

The System module allows interaction with the underlying operating system, enabling tasks such as file manipulation and environment variable access. 


import system

let current_dir = system.get_current_directory()
display(current_dir)

Creating Custom Modules

Pithon allows developers to create their own custom modules to encapsulate reusable code and extend the language's capabilities.

Steps to Create a Custom Module:

  1. Create a new `.py` file representing your module.
  2. Define functions, classes, and variables within the module.
  3. Import and use the module in your Pithon programs.

Example: Creating a Utilities Module

Here's how to create and use a simple `utilities` module.

Step 1: Create `utilities.py`

# utilities.py

function add(a, b):
    return a + b

function subtract(a, b):
    return a - b
Step 2: Import and Use the Module

import utilities

let sum = utilities.add(5, 3)
display(sum)  # Outputs: 8

let difference = utilities.subtract(5, 3)
display(difference)  # Outputs: 2
