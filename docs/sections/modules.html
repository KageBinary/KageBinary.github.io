Built-in Modules

Pithon provides several built-in modules that offer essential functionalities and constants to facilitate various operations. These modules are automatically available in your Pithon environment, allowing you to leverage mathematical and physical constants seamlessly in your programs.

Available Modules

  • Math - Provides fundamental mathematical constants.
  • Physics - Contains key physical constants.
  • System - Offers system-related constants.

Math Module

The Math module includes a collection of essential mathematical constants that are frequently used in calculations and algorithms.

Available Constants

  • pi - The mathematical constant π (~3.14159).
  • e - Euler's number (~2.71828).
  • tau - Tau, equal to 2π (~6.28318).
  • phi - The golden ratio (~1.61803).
  • gamma - Euler-Mascheroni constant (~0.57721).
  • sqrt2 - Square root of 2 (~1.41421).
  • sqrt3 - Square root of 3 (~1.73205).
  • log10e - Logarithm base 10 of Euler's number (~0.43429).

Usage Example

Below is an example of how to access and use the constants from the Math module in your Pithon script: 


# Example: Using Math Module Constants in Pithon

# Accessing Math constants
let pi = Math.pi;
let e = Math.e;
let tau = Math.tau;

# Performing calculations
let circumference = tau * radius;
let exponential = Math.e ^ x;

# Displaying results
display("Circumference:", circumference);
display("Exponential:", exponential);

Physics Module

The Physics module provides essential physical constants that are fundamental in various scientific and engineering calculations.

Available Constants

  • c - Speed of light in vacuum (299,792,458 meters per second).
  • G - Gravitational constant (6.67430 × 10⁻¹¹ m³⋅kg⁻¹⋅s⁻²).
  • h - Planck constant (6.62607015 × 10⁻³⁴ Js).
  • Na - Avogadro's number (6.02214076 × 10²³ mol⁻¹).
  • k - Boltzmann constant (1.380649 × 10⁻²³ J⋅K⁻¹).

Usage Example

Here's how you can utilize the constants from the Physics module in your Pithon script: 


# Example: Using Physics Module Constants in Pithon

# Accessing Physics constants
let speed_of_light = Physics.c;
let gravitational_constant = Physics.G;
let planck_constant = Physics.h;

# Performing calculations
let energy = planck_constant * frequency;
let force = gravitational_constant * (mass1 * mass2) / (distance ^ 2);

# Displaying results
display("Energy:", energy);
display("Force:", force);

System Module

The System module provides system-related constants that are useful for handling infinite values and floating-point precision.

Available Constants

  • inf - Represents positive infinity.
  • -inf - Represents negative infinity.
  • eps - Machine epsilon (the smallest representable difference).

Usage Example

Below is an example of how to use the constants from the System module in your Pithon script: 


# Example: Using System Module Constants in Pithon

# Accessing System constants
let positive_infinity = System.inf;
let negative_infinity = System.-inf;
let epsilon = System.eps;

# Performing comparisons
if (value == positive_infinity){
    display("Value is positive infinity.");
}
else if (value == negative_infinity){
    display("Value is negative infinity.");
}

# Checking precision
let is_precise = (difference < epsilon);
display("Is the difference within machine epsilon?", is_precise);
